# Gear Economy Research

> Skill 创作者如何在 AI Agent 经济中获利？

研究新兴的 "Agent-as-a-Service" 经济 —— 人类专业知识被封装进 Agent，通过 A2A 协议暴露，其他 Agent 付费调用，创作者从中获利。

## 核心问题

**如何构建这条价值链：人类专业知识 → Agent 服务 → A2A 交易 → 创作者收入？**

- **$19.9T** — 2030 年 Agent 经济规模（Goldman Sachs via Lightspeed）
